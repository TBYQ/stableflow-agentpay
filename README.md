# StableFlow AgentPay

StableFlow AgentPay is an Agentic AI payment infrastructure project built for Casper Agentic Buildathon 2026.

It enables AI agents to access paid real-world services through Casper Testnet transactions, payment intents, backend verification, ledger reconciliation, and signed webhooks.

## Core Flow

1. An AI agent requests access to a paid service.
2. StableFlow creates a payment intent.
3. The user or agent submits a Casper Testnet transaction.
4. The backend verifies the transaction and updates payment status.
5. A ledger entry is created.
6. A signed webhook unlocks the service.
7. AI generates a payment status and risk summary.
